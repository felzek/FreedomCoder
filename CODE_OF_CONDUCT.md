# Code of Conduct

## Our Pledge

We want FreedomCoder to be a respectful, technically honest, and welcoming open-source project.

## Expected Behavior

- Be respectful in disagreements.
- Focus criticism on ideas, code, and decisions, not people.
- Prefer evidence and reproducible claims.
- Do not exaggerate performance, safety, or model capability.

## Unacceptable Behavior

- Harassment or personal attacks
- Deliberate misinformation about the project
- Publishing private information without permission

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this code of conduct.
